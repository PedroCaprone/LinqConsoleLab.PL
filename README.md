# LinqConsoleLab - apbdCW3

Projekt konsolowy w C# - dydatkycznie pokazuje użycie LINQ:
Filtrowanie danych przy użyciu Where,
Projekcja danych przy użyciu Select,
Sortowanie za pomocą OrderBy i ThenBy,
Pobranie pojedynczego elementu przy użyciu FirstOrDefault,
Sprawdzenie warunku z użyciem Any i All,
Zliczanie rekordów przez Count,
Usuwanie duplikatów dzięki Distinct,
Przygotowanie prostego stronicowania przez Skip i Take,
Łączenie danych z użyciem Join,
Spłaszczanie danych za pomocą SelectMany,
Grupowanie rekordów przy użyciu GroupBy,
Podstawowe agregacje, takie jak Average i Max,

Zadania 1–16 użycie Linq bazując na poleceniach sql
Zadania 17-20 - wyzwania, łącza wiele umiejętności

## Uruchomienie
Uruchom Program.cs i wybierz opcję 0 - 20 z konsolowego menu.
